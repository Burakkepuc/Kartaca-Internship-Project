Tebrikler Burak Kepuc, gorev icerigine ulastin!

Iste bekledigimiz gorevin gereksinimleri;

Tercih ettiginiz bir arka uc dili (tercihen Java / Go / PHP / Python) ve on yuz js catisi ile (tercihen Vue / React / Angular)

Temel seviyede kullanici uyeligi yapilabilen, oturum acma ve kapatma ozelliklerine sahip bir web projesi gelistirmesi beklenmektedir.

Oturum acan kullanicilar anasayfa uzerinde bir takvim gormelidir. Oturum acmamis olanlar ise oturum acmaya yonlendirilmelidir.
Takvimde kullanicinin kendi ekledigi etkinlikler gosterilmeli, takvime tiklayarak ekleme yapabilmeli veya etkinligi kaldirabilmelidir.

Arka uc 3 micro service seklinde tasarlanmalidir. Account service / Calendar service / Web API service.

Oturum bilgileri icin Redis kullanilmalidir. RDBMS veya NoSQL tercihi istege gore yapilabilir.

Tum on yuz istekleri Web API service tarafindan karsilanmalidir ve servisler kendi aralarinda konusmalidir.
